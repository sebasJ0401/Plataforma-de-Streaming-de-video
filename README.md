# Plataforma-de-Streaming-de-video
Proyecto de diseño de patrones de software
Johan Sebastian Jaimes Burbano
Alejandra Mayorga Barboza


Alcance del Proyecto

El proyecto incluye:
Gestión de contenido (subir, editar y eliminar videos)
Gestión de usuarios (registro e inicio de sesión)
Sistema básico de suscripciones
Recomendación simple de contenido
Streaming en diferentes calidades
Protección de contenido (DRM simple)

Funcionalidades:
Gestión de Usuarios
Registro
Inicio de sesión
Perfil básico
Validación de acceso según suscripción

Gestión de Contenido
Subir videos
Clasificarlos por categoría
Mostrar lista de videos disponibles

Suscripciones
Plan gratuito (acceso limitado)
Plan premium (acceso completo)

Recomendaciones
Sistema sencillo basado en:
Videos más vistos
Categorías favoritas del usuario

Streaming Adaptativo
Videos en 480p y 720p
Selección manual de calidad

DRM 
Restricción de descarga directa
Acceso solo para usuarios autenticados

Tecnologías:
Backend: Python
Frontend: HTML + CSS 
Base de datos: MySQL o PostgreSQL
Reproductor: HTML5 Video Player
